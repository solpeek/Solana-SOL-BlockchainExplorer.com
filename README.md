# SolPeek Explorer

A Solana Blockchain Explorer website for tracking gas fees and refunds for crypto traders.

## Features

- **Solana Network Stats**: Display real-time SOL price, total supply, current slot, epoch, and transactions per second
- **Token Search**: Search for Solana tokens by address
- **Network Performance**: View epoch progress and TPS charts
- **Recent Blocks & Transactions**: Browse recent blocks and transactions on the network
- **Wallet Integration**: Connect Phantom Wallet for donations
- **Donation Support**: Contribute SOL to support the project

## Getting Started

1. Clone the repository
2. Open `index.html` in a web browser
3. No build process required - it's a static frontend!

## Technologies Used

- HTML5
- CSS3 (with Solscan-inspired design)
- JavaScript (Vanilla)
- Solana Web3.js Library
- CoinGecko API for SOL pricing

## Features

### Dashboard Stats
- Real-time SOL/USD price

- Current network slot
- Current epoch number

- Recent blocks table
- Recent transactions table
- Epoch progress bar

### Wallet & Donations

- Quick donation amounts (0.01, 0.05, 0.1 SOL)
